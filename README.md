# mini-grading-system

Purpose: practicing STL algorithms + iterators.

```
transform, find_if, accumulate, partition_copy, sort, stable_partition...
-More in the future (Inshallah/إن شاء الله)-
```

How to build/run: 

With make file:

```
make
```

Manually:

```
g++ -std=c++20 main.cpp -o main 
```
Then:

```
./main.exe # Windows
./main     # Linux
```
